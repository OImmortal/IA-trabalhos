Software + Problema: O GitHub Copilot (Desktop/Web) resolve o problema do desenvolvedor gastar muito tempo escrevendo código repetitivo ou pesquisando documentações de sintaxes específicas.

Funcionalidade de IA + Entrada/Saída: Atua como um programador, prevendo e sugerindo blocos inteiros de código ou funções em tempo real na IDE.

Entrada: O contexto do código atual, arquivos abertos, nomes de variáveis e comentários em linguagem natural (ex.: // criar função para validar CPF).

Saída: Trechos de código gerados e exibidos diretamente no editor como texto sugerido.

Técnica provável: LLM (Large Language Model, especificamente modelos baseados na arquitetura GPT focados em código, como o OpenAI Codex).

Métrica de sucesso: Taxa de aceitação (a porcentagem de vezes que o usuário pressiona Tab para aceitar a sugestão em relação ao total de sugestões exibidas) e redução no tempo de entrega (lead time) das tarefas de desenvolvimento.
